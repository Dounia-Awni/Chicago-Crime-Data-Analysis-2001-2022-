## 📊 Tableau Dashboard — Chicago Crime Analysis

This interactive Tableau dashboard visualizes 22 years of Chicago crime data (2001–2022), 
providing insights into crime trends, arrest rates, rush hour patterns, and seasonal behavior.

### 📋 Dashboard Features

**Main Dashboard — Chicago Crime Overview:**
- 📉 Crime Trend Over Years (2001–2022) — shows the 51% overall decline
- 🥧 Arrest Rate — breakdown of arrested vs non-arrested cases
- 🕐 AM vs PM Rush Hour — crime comparison across rush hour windows
- 📊 Top 10 Crime Types — most common crimes in Chicago

**Secondary Dashboard — Crime Deep Dive:**
- 🗓️ Monthly Crime Heatmap — seasonal patterns across all years

### ⚙️ Interactive Features
- Click any point on the Crime Trend line chart to filter 
  Top 10 Crime Types and AM vs PM charts by year
- Navigate between dashboards using the built-in buttons
